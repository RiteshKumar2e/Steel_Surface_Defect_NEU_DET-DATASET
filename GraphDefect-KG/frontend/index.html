Model pipeline

How an image becomes an explained prediction.

  1. Preprocess — resize, CLAHE, edges, patches, regions
  2. Extract — MobileNetV2 + 32 handcrafted descriptors
  3. Graph — KNN region graph over patch nodes
  4. Learn — GCN + GAT message passing, gated fusion
  5. Reason — knowledge graph + evidence explanation
